# Bike Shop SQL Data Analysis

🔗 **Related Dashboard Project:** [View Tableau Dashboard](PASTE_TABLEAU_REPO_LINK)

---

## Overview

This project focuses on the data preparation and analysis behind a retail sales dashboard. It includes data cleaning, transformation, feature engineering, and aggregation to generate business insights.

The goal was to structure raw consumer spending data into meaningful datasets that support business decision-making and visualization.

---

## Objectives

- Clean and structure raw consumer spending data  
- Create derived features such as profit margin and age groups  
- Aggregate data for business insights  
- Prepare datasets for Tableau dashboard development  

---

## Data Pipeline

Raw Data → Excel Cleaning → SQL Analysis (Notebook) → Aggregated Outputs → Tableau Dashboard

---

## Key Steps Performed

### Data Cleaning
- Removed inconsistencies and missing values  
- Standardized column formats  
- Prepared dataset for analysis  

### Feature Engineering
- Created profit margin calculations  
- Grouped customers into age segments  

### Aggregations
- Product-level profitability analysis  
- Customer segmentation by age group  
- Revenue trends over time  

---

## Outputs

- `product_performance.csv` → Product profitability analysis  
- `age_analysis.csv` → Customer segmentation  
- `time_trend.csv` → Revenue trends over time  

---

## Tools Used

- SQL (within Jupyter Notebook)
- Python (data handling)
- Excel (initial cleaning)
- Tableau (visualization)

---

## Project Structure

- `/data/raw` → Original dataset  
- `/data/cleaned` → Cleaned dataset

- `/outputs` → Final datasets used in Tableau
